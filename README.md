# lol-today — 칼바람 딜량 내기 정산기

롤 LCU API로 최근 칼바람 전적을 불러와, 친구 3~5명이서 건 딜량 내기를 자동 정산하는 로컬 웹 도구.

## 정산 룰
- 3인: 꼴등 → 1등 3,000원
- 4인: 꼴등 → 1등 3,000원 / 3등 → 2등 1,000원
- 5인: 꼴등 → 1등 3,000원 / 4등 → 2등 1,000원

## 설치 & 실행
```bash
pip install -r requirements.txt
python -m cli
```
브라우저가 자동으로 `http://127.0.0.1:8765/` 열림.
롤 클라이언트가 켜져 있어야 동작.

### 설치 경로 커스텀
기본 경로에 롤이 안 깔려있으면 환경변수:
```bash
set LCU_INSTALL_PATH=D:\Games\League of Legends
```

## 사용 흐름
1. 최근 매치 리스트에서 정산할 칼바람 판을 체크
2. "정산 준비" → 매치별로 친구 체크박스 조정 (풀에 없는 팀원은 "친구풀에 추가")
3. "정산 실행" → 매치별 순위/지급 + 세션 전체 지급 매트릭스 + 복붙용 텍스트 요약

## 친구 풀
`config/friends.json`에 저장. 포맷:
```json
{
  "friends": [
    {"game_name": "미르", "tag_line": "KR1", "puuid": "..."}
  ]
}
```
웹 UI "친구풀 추가" 버튼으로 자동 갱신되거나 직접 편집 가능.
